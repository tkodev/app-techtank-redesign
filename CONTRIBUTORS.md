# Contributors

Thank you to everyone who has contributed to the TechTank TO website.

## How to contribute

1. Fork the repository and create a branch from `main`.
2. Follow the [Conventional Commits](https://www.conventionalcommits.org/) format for commit messages.
3. Open a pull request using the PR template and describe what you changed and why.
4. Keep changes focused — one concern per PR.

For questions, reach out at [techtankto@gmail.com](mailto:techtankto@gmail.com).
